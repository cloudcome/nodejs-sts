#sts [![NPM version](https://img.shields.io/npm/v/sts.svg?style=flat)](https://npmjs.org/package/sts)
STATIC SERVER => sts


# INSTALL
```
npm install sts -g
```

# USAGE
```
cd your directory
sts 18080 your server name
```
* A HTTP static server will run in this directory.
* `18080` **required**
* `your server name` **optional**
