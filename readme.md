#sts [![NPM version](https://img.shields.io/npm/v/sts.svg?style=flat)](https://npmjs.org/package/sts)
STATIC SERVER => sts


# INSTALL
```
npm i sts -g
```

# USAGE
```
cd your directory
sts 18080 your server name
```

* HTTP静态服务器将运行在当前目录。
* 端口，必须: `18080`
* 名称（防止在cmd窗口运行多个静态服务器，不知道哪个是哪个），可选: `your server name`
